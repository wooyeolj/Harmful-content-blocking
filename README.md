# 주요 기능
1. 실시간 유해요소 탐지 (YOLOv5)

   객체 탐지 모델 : YOLOv5 커스텀 모델
   탐지 가능한 유해요소 : [칼, 담배, 총기]
   실시간 처리 지원

2. 메신저 앱 연동
   동영상 전송 시 필터링 수행
   - 전송 전 서버에서 실시간 프레임 분석
   - 유해 요소 탐지 시 즉시 차단

3. 보고서 생성

   발견된 유해요소 종류 및 정확도
